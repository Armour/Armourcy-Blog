耶！入手Macbook pro～

爱生活
  • Google Chrome
  • Mac
  • Xcode
  • autojump
  • iTerm2
  • markdown
  • shell
  • zsh
  • 买买买

很早之前就想入手一台mac 方便ios开发 也算是给自己开一个新的坑掉进去w

寒假和父母商量了下于是便在教育商店买了一天mac，从下单到邮寄到手大概6天，货到付款，包装也挺精美，感觉还是挺不错的～

以下是窝的mac的参数信息：

入手之后选择了英文语言启动 然后开始着手进行各种配置和常用功能部署：

在Appstore下载了Xcode

因为习惯了使用google chrome所以下载了下来并且配置好了翻墙（不过总感觉mac下goagent翻墙和vps翻墙效果都没windows好？ 感觉貌似经常断。。）

下载了Teamviewer并和自己的windows机连了起来 方便以后远程和传文件

选择使用Parallels desktop这款虚拟机，顺便试用了一下内置的免费win10。。

然后下了QQ。。。 毕竟我还在国内。。

为了使各种文档方便查询 加快开发进度 安装了Dash

安装使用了Alfred这款神器 极大的加速了我寻找各方面资料的速度

安上了进行markdown编辑的Mou，据说mac下这货不错

对了还安上了 Evernote 和 Dropbox~

最后回头一看 妈蛋命令行好丑不能忍。。于是去网上参考了一下，自己动手配置好了 zsh(agnoster themes) + iTerm2 + autojump + git 版本控制型的终极shell ＝w＝

总结一下 入手mac后基本没用鼠标。。。 触摸板功能简直不能更赞！

retina屏幕也是不错

多屏幕切换也是比windows好的多（尤其是装了虚拟机的情况下）

总之，用了mac之后感觉整个人都优雅了起来2333

（附上一些mac下的小技巧：

空格键快速预览

触发角快速锁屏

触摸板可以设置成触摸板轻敲替代按下

浏览器用safari 或 google chrome

选中一个文件按 enter 可以直接改名

改后缀名：右键，显示简介

shift+option， 可以4分1格调节声音大小（这个同样适用键盘背光和屏幕亮度）

terminal: say “blahblah” Option+Esc 直接朗读

Command＋shift＋3:全屏截图，保存截图到桌面

Command＋shift＋4:鼠标选定区域截图，保存截图到桌面

(使用Command＋shift＋4后，按下空格键，鼠标会变成一个小相机，这时候你使用鼠标对一个窗口点击一下鼠标左键，你已经对一个窗口进行了截图。)

(按Command＋shift＋4 后 ,画一个抓取的区域，不要松开鼠标，接着

1. 按住空格可以移动这个区域

2. 按住 Shift后，将锁定X 或者 Y轴进行拖动

3. 按住 Option后 将按照区域圆心进行放大.)

将bash换成zsh 使用agnoster主题 +iTerm2 +autojump +git => 终极shell

cmd-n new windows

cmd-w close windows
