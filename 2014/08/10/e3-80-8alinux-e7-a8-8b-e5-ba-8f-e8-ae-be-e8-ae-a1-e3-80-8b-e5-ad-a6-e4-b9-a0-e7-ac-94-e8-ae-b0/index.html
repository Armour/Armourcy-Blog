《Linux程序设计》学习笔记

代码宅
  • C
  • Linux
  • shell
**基本命令:**
    修改[当前用户]密码:        passwd [xxx]
    访问在线手册:            man xxx
    [当前]主机名字:            hostname [xxx]
    显示[修改]时间:            date [xxx]
    挂载以及卸载:            mount [-t vfstype][-o options] device dirname
                        -    mount -t vfat /dev/sda1 /media/external -o uid=1000,gid=100,utf8,dmask=027,fmask=137
                        -    mount -t ntfs -3g /dev/sda1 /media/external
                        -    umount    media/external    
    find
    文件打包:                tar [options][filename-list]
                        -    tar -cvf help.tar *.help
                        -    tar -zxvf help.tar.gz
    [设置]缺省文件权限:        umask [xxx] 
    硬链接:                ln file file.hard
    符号链接:            ln -s file file.soft
**shell编程:**
    $0                shell脚本的名字
    $* / $@                    命令行中的所有参数
    $#                命令行参数的总个数
    $$                shell脚本进程的ID号
    $?                最后一次命令的退出状态
    $!                最近一次后台进程的ID号
    命令检测语法:                    test [expression]
                    [[expression]]
                    字符表达式:
                        s1 = s2   相等
                        s1 != s2  不等
                        -n s1     不为空
                        -z s1     为空
                    算术表达式:
                        e1 -eq e2 相等
                        e1 -ne e2 不等
                        e1 -gt e2 大于
                        e1 -ge e2 大于等于
                        e1 -lt e2 小于
                        e1 -le e2 小于等于
                        ！e1       为假
                    文件测试表达式:
                        -d file   目录
                        -e file   文件存在
                        -f file   普通文件
                        -r file   可读
                        -w file   可写
                        -x file   可执行
                        -s file   长度不为0
    if语句格式:        if-then-elif-else-fi
    for语句格式:            for [variable in xxx]-do-done
    while语句格式:            while-do-done
    until语句格式:            until-do-done
    case语句格式:        case variable in
                    pattern1 ) command-list1 
                        ;;
                    pattern2 ) command-list2
                        ;;
                    ...
                    patternN ) command-listN
                        ;;
                    esac
    expr格式:        `expr $x + $y`  
                $(expr $x + $y ) 
    数组
**Linux下C语言开发:**
    C编译:            gcc -o hello hello.c             ##直接产生可执行文件
                gcc -c program.c pro1.c pro2.c         ##产生目标文件 *.o
                gcc -o program program.o pro1.o pro2.o     ##连接多个*.o产生可执行文件
                ar crv libfoo.a pro1.o pro2.o         ##创建一个库文件
                gcc -o program program.o libfoo.a     ##使用一个库文件产生可执行程序
    Makefile:            
                OBJS = prog.o code.o
                CC = gcc
                test: ${OBJS}
                    ${CC} -c test $ {OBJS}
                proh.o: prog.c prog.h code.h
                    ${CC} -c prog.c -o prog.o
                    code.o: code.c code.h
                    ${CC} -c code.c -o code.o
                clean:
                    rm -f *.o
    gdb调试工具:            gcc -g hello.c -o hello
                gdb -q hello
                可用命令:                
                    l (list)        ##查看源代码 
                     b (break)        ##设置断点
                    info b (info break)    ##查看断点信息
                    r (run)            ##运行程序
                    p 变量名        ##查看变量值
                    n (next)        ##单步运行（不进入函数
                    s (step)        ##单步进行（进入函数
                    c (continue)        ##继续执行完程序
                    q (quit)        ##退出
